Multiplying Binomials x Trinomials

- The product of a binomial and a trinomial is another sort of polynomial multiplication problem. Although the FOIL method cannot be implemented since a trinomial has more than two terms, the Distributive Property is still used to arrange the individual products. Each term in the binomial must be multiplied by each term in the trinomial using the distributive property.

- Let (a + b) be a Binomial and (c + d + e) be a Trinomial. Product of those two expressions, = (a + b)(c + d + e) = ac + ad + ae + bc + bd + be. It is a Polynomial.

• Example 1: Find the product. (2x + 4) and (3x^2 + 6x + 5)

Step 1: Distribute the trinomial among the binomial terms:

2x(3x^2 + 6x + 5) + 4( 3x^2 + 6x + 5)

Step 2: Distribute the monomials to each term in the trinomials using the distributive property.

2x(3x^2) + 2x(6x) + 2x (5) + 4(3x ^2) + 4(6x) + 4(5)

Step 3: Multiply

6x^3 + 12x^2 + 10x + 12x^2 + 24x +20

Step 4: Combine like terms

6x^3 + 24x^2 +34x +20

• Example 2 (With Subtraction): Find the product. (4x + 1) and (2x^2 - x + 2)

Step 1: Use the distributive property:

4x(2x^2 - x + 2) + 1 (2x^2 - x + 2)

Step 2: Multiply:

(8x^3 - 4x^2 + 8x) + (2x^2 - x + 2)

Step 3: Combine like terms:

8x^3 + (-4x^2 + 2x^2) + (8x - x) + 2

Step 4: Simplify

8x^3 - 2x^2 + 7x +2
